 CyberBotSA Part 2 - GUI Cybersecurity Awareness Chatbot

Description
A WPF C# application that provides a graphical cybersecurity awareness chatbot with sentiment detection, memory recall, and dynamic responses.

Features
- WPF GUI with dark theme
- Voice greeting on startup
- Keyword recognition for cybersecurity topics
- Random responses for varied interaction
- Sentiment detection
- Memory and recall of user details
- Conversation flow with follow up support
